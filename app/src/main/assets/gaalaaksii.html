Gaalaaksii

Galaxy

Seensa

Gaalaaksiin gurmii urjiilee, gaazota, dhukkee/ awwaara(dust)fi maatara dukkanaa'aa (Dark Matter)iiti.Jechi gaalaaksii jedhu kun jecha afaan Giriikirraa kan dhufe yoo ta'u afaan Griikiin "galaxya" jedheme beekkama. Hiikni isaas qaama aannan faakkaatu (Milky) jechuu dha.

Gaalaaksonni urjiileerraa waan ijaarramaniif qaamolee kana giddu humna harkisaa guddaa ta’tetu dalageeffama. Yeroo hawaan keenya balachaa deemuss gaalaakisoonni boca isaani hingadhiisanu.


fakkii 1 Gaalaaksii miilkii weey

Ibsa

Gaalaakisoonii hanga, qabeefii boca adda addaatiin argamu. Gaalaaksiwwaan xixiqqoo kan jedhaman (Dwarf galaxy) urjilee naannoo 107 kan qaban yoo ta’u isaan gurguddoo ykn giant jedhamanimmoo urjilee naannoo 1014 qabatanii argamu.

Akkkuma Lafti keeenya Aduu marsitee argamtu urjiileen gaalaaksii keessatti argamanis handhura galaksii kan marsan yoo ta’uu handhuurri gaalaakisoota baayye Holqa gurraacha(Black hole) guddda of keessatti qabata. Gaalaakisuma keessaatti urjiileeen gurmiii kan uuman yoo ta’uu naannon kuun naannoo “intlstellar cloud” jedhama.

Akkuma gurmiin urjii gaalaksii uumu gurmiin gaalaaksiiwwani immoo Klasstara (Galactic cluster) uuma. Maqaan gaalaaksii nuti keessa jirruu Miilkii weey (milky way)yoo ta'u kan argamus klaastara local gruop jedhamu keessa. Qaamni gaalaaksii keessa jiru tokko yeroo hunda sochiirra jira.


fakkii 2 Kilaastara Gaalaakisota

Gosoota

Gaalaaksiiwwan hammaa fi gosa adda addaatin faca'anii jiran.Hammi isaaniis gaalaaksiiwwan xixiqqoo (dwarf) irraa kaasee amma isaan gudguddaa (giant)jiran ta'uu danda'a.

Gaalaaksiiwwan xixiqqoo kan jennuun urjjiiwwan miilyoona kudhanii gadi kan qaban yoo ta'u isaan gudguddoon immo urjii hanga triilyoona dhibbaa (100,000,000,000,000)qabaachuu danda'u.


fakkii 3 Gaalaaksii Diwaarfii

Akkuma duraan kaasnee tuurreee kilaastarri gaalaaksii gaalaaksiiwwan hedduun walitti dhufuun kan uummamu yoo ta'u Klaastarri urjii immoo urjiileen walitti dhufanii kan uumanidha.

Fakeenyaaf:- Klaastarri Gloobuulaar klaastara dhuma gaalaaksii keenyaa (milky way) irratti kan argamuudha.

Gara kilaastara gaalaaksii yeroo debinu, Gaalaaksiiwwan kan naan'an birqaba/relatively gaalaaksota biraa waliini.Yeeoo tokko tokko garuu gaalaaksoonni lama walittu bu'uu danda'u.Yeroo walitti buan kana garuu guutumaan guutuutti kan badan osoo hinataanee yaadota armaan gadii sadeen keessaa yoo xiqqaate illee tokkotu iddoo fudhata.

    1. Urjiileen walbaleessuu (past each other)godhu.

    2. Gaazonni duumeessawoo(Gas cloud) fi dhukkeen (dust) walitti makamuun urjiilee haara'a uumu.

3.Urjiileen haara'a uummaman kunnen immoo walitti dhufuun gaalaaksii haara'a uumu.

1. Gaalaaksii dambalawoo (spiral galaxy)

1. Gaalaaksii dambalawoo(Spiral_galaxy)
Gaalaaksiin kun bifa "spiral" ykn spriingii kan qaban yoo ta'u gostii gaalaaksii kun hawaa keenya keessatti argamuudhan sadarkaa 1ffaa irra jira. Qabiyeen isaatis 77% dha.

Gosti gaalaaksii kun matuma isaatin bakka lamatti qooddama

      A. "Barred spiral galaxy" -gaalaaksii gissuun isaa bifa reektaangila fakkaatu qabu fi

      B. "Unbarred spiral galaxy"- Gaalaaksii gidduun isaa bifa gewngoo qabudha.

Gaalaaksiiwwan dambalawoo ykn "spiral" gaalaaksiiwwan biraa irraa maaltu adda isaan godhe? Bifa diiskii qabaachuu isaanii,irree gaalaksawaa (spiral arm) fi gidduun isaanii urjiiwwan dulloomaniin guuttamuu isaati.

"Spiral Arm" kan jennun iddoo urjiin itti dhalatu too ta'u bifa duumeessaa qaba.
