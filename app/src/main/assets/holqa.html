Holqa Gurraacha

Black hole

Seensa

Holqi gurraachi naannoo Hawaa iddoon itti ifti ilee jalaa milquu hin dandeenyedha.Holqi gurraachii walumaa galatti qaamolee 5 kan qabu yoo ta’u qaamoolee babbekkamoo babbeekkamuu kan jedhaman “Event Horizon” fi “Singularity” dha. Inni jalqabaa naannoo qaamoollee Holqa kana marsanii jiran bocnii isaanii harkifamee fi dheeratee itti argamudha. Naannoon inni lammafaanimmoo “point of Singularity” kan jedhamu yoo ta’u handhuura Holqa gurraachaatii. Qaamni naannooo kana ga’ee deebiee bahu hin danda’u.

Holqi Gurraachii ifa gara isaatti dhufe gutumaan guutuutii oftti fudhata. Sababa kanaaf kallattiidhan Teeleeskooppii Optikaalaatiin muullachuu hin danda’u. Haat’au malee dhiibbaa fi harkisa naannoo isaatti gaggeessuun adda baafachuun ni danda’ama.

Qaamoleen “Event Horizon” keessa seenan deebianii ba’uu hin danda’anu.Sababni inni tokkoffaan harkisni isaa gudda waan ta’eef yoo ta’u inni lammafaan imo Teempireecharri naannoo kanaa olaanaa wan ta’eefidha. Kanaafi Holqi Gurraachi faakkiirratti naannoon isaa ifaa kan fakkaatu.


fakkii 1 Holqa Gurraacha

seenaa

Yaaddni Holqa gurraachaa yeroo duraatif kan ka’e bara 1783 nama “John Michell” jedhamuun turee. John miikeel yeroo baayyee qorannoo Harkisa Qaamotaarratti Qoranno gaggeessaa kan ture yoo ta’u, hanga qaamni tokko guuddachaa deemu harksini isaas dabalaa akka deemu hubate. Kanarraan kan ka’e qaamni guddaan hawaa keenya kessatti amala akkaanaa qabaachuu malu akka jiru dubbatee ture.

Bara 1916 “Albert Einstein” Tiyoorrii beekkamaa isaa kan “General Relativity” jedhu maxxansiise akka Albert jedhutti

• Hangi (mass) qaamnii tookko hanguma dabalaa deemu naannoo isaatti harkisa olaanaa dalageessuu jalqaba. Jecha biraatin wanta spacetime jedhamu gara ofiitti dabsaa (bend) gochaa deema. Ifti naanno kana ga’es akkuma qama biro dabaa deema. Kanaan Humna harkisaa jannee waamna.

• Ifti naanoo humni harkisaa isaa olaanaa ta’e ga’emmo saffisni isaa jijjiirramuu dand’a.

Yaada “Albert Einstein” irraa ka’uun yoo ifti naannoo humni harkisaa isaa kan nu yaadnu caalaa ta’ee ga’e maal akka isa mudatuu tilmaamuun ni danda’ama. Ji’oota muuraasa boda yeroo waraana adunyaa jalqabaa Fiiziistiin biyya Jarman “Karl Schwarzschild” foormulaa “Albert Einstein” fayyadamuudhan dhuguma naannoo kun jiraachuu akka danda’u ibse. Raadiyasiin Handhuura Holqa gurraachaa irraa fi naannoo “Event Horrizon” giddu jirus “Schwarzschild radius” jedhamuun beekkama.


fakkii 2 “Schwarzschild radius”

Bara 1930 Astiroonoomariin “Subrahmaniyan Chandrasekhar” urjiileen Aduu keenya irra guddatan marsaa jireenyaa isaanii gara dhumaatti boba’aa isaannii (hydrogen gas) fixachuun du’uu akka danda’aan qorannoon bira ga’e. ittii aansuun bara 1939 “Robert Oppnheimer” urjiin tokko Holqa gurraacha uumuuf yoo xiqaatee Hangi isaa dacha 3 aduu keenya caaluu akka qabu hime. Bara 1967 namni “John Wheeler” jedhamu jecha “Black Hole” jedhu uume. Isa dura urjiile gurraacha jedhamanii beekkamu ture.

gosoota holqa gurraachaa

Hamma isaaniirratti hundaa’uun holqa gurraacha (black hole) bakka afuritti qoodne ilaaluu dandeenya. Isaanis

1. Holqa gurraacha isa guddaa (super massive black hole)
Isa guddaa kan ta’eef yeroo baayyee gidduu gaalaakistotaa keessatti kan argamudha.


fakkii 3 Holqa gurraacha isa guddaa
Gidduu miilkiiweey keessatti argamu

2. Holqa gurraacha isa giddu galeessaa (intermediate black holes )
Ramaddiin kun Holqeen gurraacha Hangii isaanii aduu keenya dacha 100 hanga dacha 1000 000 kan caalanidha.


fakkii 4 Holqa gurraacha isa giddu galeessaa

3. Holqa gurraacha urjiilee (stellar black holes)
Holqeen gurrachii kunimoo yeroo urjileen gurguddaan dua’an yoo ta’u angi isaanii hanga aduu keenyaa dacha 5 hanga 100 kan ta’anidha.


fakkii 5 Holqa gurraacha Urjiilee

4. Holqa gurraacha isa xiqqaa ( micro black holes)
Gosti holqa gurraacha kun kan ibsame kosmoloojistii beekama “Isteefan Haawkingiin” bara 1971 tti ture innis qabeen isaanii hamma atoomii tokkoo ta’u kan danda’uufi hangi isaanii hamma ija namaa gahuu akka danda’udha.


fakkii 6 Holqa gurraacha isa xiqaa

Qaamolee Holqa Gurraachaa

Holqi gurraachii qaamolee babbeekkamoo 5 irraa ijaarrame.

1. “Event horizon”
Naannoon kuun Holqa gurraacha keessaa isa beekkamaa yoo ta’u daangaa holqa gurraachaati.

2. “Singularity”
Handhuura Holqa gurraachaa yoo ta’u ijaan hin muullatu.

3. “Photon Sphere”
Naannoon Fottoonoonni ifaa itti argaman yoo ta’u furdina hin qabu.

4. “Ergosphere”
Naanon kuun naanoo qaamooleen yeroo hunda sochiirratti aragamanidha.

5. “ISCO”/innermost stable circular orbit

Naannoo kanatti sudoowwan (particles) faffac’anii saffisa walfakkaataan iddoo itti argamanidha.
